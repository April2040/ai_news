# 🌍 AI信息聚合平台 - 本地部署指南

## 📋 问题诊断

**原问题**: 在云端环境中无法访问 localhost:8081  
**原因**: 云端环境的网络限制，localhost服务器无法被外部访问  
**解决方案**: 提供本地部署方案

## 🚀 本地部署方案

### 方案1: 使用启动脚本 (推荐)

#### Windows用户
1. 下载所有项目文件到本地文件夹
2. 双击运行 `start_server.bat`
3. 浏览器会自动打开 http://localhost:8081

#### Mac/Linux用户
1. 下载所有项目文件到本地文件夹
2. 打开终端，进入项目目录
3. 运行命令: `chmod +x start_server.sh && ./start_server.sh`
4. 浏览器访问 http://localhost:8081

### 方案2: 手动启动

#### Windows
```cmd
# 打开命令提示符或PowerShell
cd "你的项目文件夹路径"
python -m http.server 8081
```

#### Mac/Linux
```bash
# 打开终端
cd /path/to/your/project
python3 -m http.server 8081
```

### 方案3: 使用Node.js
如果你安装了Node.js：
```bash
npx http-server . -p 8081
```

### ❌ 常见错误解决

#### 错误1: "Python不是内部或外部命令"
**解决方案**: 
1. 安装Python3: https://www.python.org/downloads/
2. 安装时勾选"Add Python to PATH"

#### 错误2: "端口8081已被占用"
**解决方案**: 
1. 关闭其他可能占用端口的程序
2. 或者修改端口: `python -m http.server 8082`

#### 错误3: "页面显示空白或样式异常"
**解决方案**: 
1. 检查所有文件是否完整下载
2. 确保文件夹结构正确
3. 清除浏览器缓存后重试

#### 错误4: "无法访问此网站"
**解决方案**: 
1. 确认服务器已启动
2. 尝试访问 http://127.0.0.1:8081
3. 检查防火墙设置
4. 尝试使用其他端口

## 🔧 高级配置

### 修改端口
如果8081端口被占用，可以修改为其他端口：
```bash
python -m http.server 3000  # 使用3000端口
```

### 指定网络接口
默认绑定到所有接口(0.0.0.0)，如果需要只绑定本地：
```bash
python -m http.server 8081 --bind 127.0.0.1
```

### 使用HTTPS (可选)
如果需要HTTPS支持，可以使用Node.js的http-server:
```bash
npx http-server . -p 8081 --ssl
```
