## 去重与质量控制策略

### 1. 标题相似度去重
```python
from difflib import SequenceMatcher

def is_duplicate(title1, title2, threshold=0.85):
    """检查两个标题是否重复"""
    ratio = SequenceMatcher(None, title1.lower(), title2.lower()).ratio()
    return ratio > threshold
```

### 2. 时效性过滤
- 只保留最近7天内的新闻
- 超过24小时的新闻降低重要性评分

### 3. 内容质量评分因子
| 因子 | 权重 | 说明 |
|-----|------|------|
| 源权威性 | +2.0 | 头部源加分 |
| 标题含公司名 | +1.5 | OpenAI、Google、Anthropic等 |
| 标题含动作词 | +1.0 | 发布、突破、收购等 |
| 内容长度 | +0.5 | 摘要>100字加分 |
| 有作者信息 | +0.5 | 非匿名文章 |
| 重复内容 | -3.0 | 与已有新闻重复 |

### 4. 每源数量限制
| 优先级 | 每源最大条数 | 说明 |
|-------|-------------|------|
| 高 | 15条 | 头部权威源 |
| 中 | 10条 | 专业技术源 |
| 低 | 5条 | 社区动态源 |

## 注意事项

1. **请求频率**：每个源间隔1-2秒，避免被封禁
2. **User-Agent**：使用合理的浏览器UA
3. **错误处理**：单源失败不影响整体
4. **缓存机制**：避免重复抓取相同内容
5. **合规性**：遵守robots.txt和使用条款
